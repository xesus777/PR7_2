### Вариант 12: ROT13
Метод: частный случай шифра Цезаря со сдвигом 13.
Задача: реализовать симметричный метод Rot13(string text).
Требования:
- самодвойственность (шифрование = дешифрование);
- только для латинского алфавита.
Тестирование:
- проверка самодвойственность;
- игнорирование неалфавитных символов.

---

## Анализ предметной области

Описание предметной области:
- ROT13 - симметричный шифр подстановки для латинского алфавита
- Сдвиг на 13 позиций (половина алфавита из 26 букв)
- Самодвойственность: ROT13(ROT13(text)) = text
- Неалфавитные символы (цифры, знаки, пробелы) не изменяются
- Регистр букв сохраняется

Функциональные требования:
1. Шифрование текста латиницы методом ROT13
2. Дешифрование текста (совпадает с шифрованием)
3. Валидация входных данных (null, пустая строка, кириллица)
4. Обработка исключительных ситуаций

Нефункциональные требования:
1. Графический интерфейс с полями ввода/вывода
2. Всплывающие подсказки (ToolTips)
3. Интуитивно понятное управление (кнопки)
4. Устойчивость к некорректному вводу
